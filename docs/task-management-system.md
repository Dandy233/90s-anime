# 智能任务管理系统

## 概述
这是一个基于 OpenClaw 的智能任务管理方案，结合定时任务、记忆系统和飞书通知，实现自动化任务跟踪和提醒。

## 组件

### 1. 记忆系统（已配置）
- **文件**: `memory/YYYY-MM-DD.md` - 每日日志
- **文件**: `MEMORY.md` - 长期记忆和待办
- **功能**: 记录任务、事件、观察

### 2. 定时任务（已配置）
| 任务 | 时间 | 功能 |
|------|------|------|
| daily-it-news | 09:00 | 早间新闻简报 |
| daily-task-review | 18:00 | 晚间任务回顾 | 运行中（偶发飞书发送失败）|
| weekly-system-check | 周一 10:00 | 系统健康检查 |
| auto-git-sync | 每小时 | 代码自动同步 |

### 3. 心跳检查（已配置）
- **频率**: 每30分钟
- **功能**: 检查待办事项和系统状态
- **配置**: `HEARTBEAT.md`

### 4. 飞书集成（已配置）
- **渠道**: 飞书机器人
- **功能**: 消息推送、交互
- **技能**: feishu-doc, feishu-drive, feishu-wiki

## 工作流程

### 早晨 (09:00)
1. 接收 IT 新闻简报
2. 查看今日待办
3. 开始一天工作

### 白天
1. 每30分钟心跳检查
2. 随时记录任务到 memory/
3. 使用飞书与我交互

### 晚间 (18:00)
1. 接收任务回顾日报
2. 查看今日完成情况
3. 规划明日任务

## 使用方法

### 添加待办
直接告诉我："记住我要做XXX"
我会更新 MEMORY.md

### 查看任务
问："我今天有什么任务？"
我会检查 memory/ 和 MEMORY.md

### 修改任务
说："把XXX任务改成已完成"
我会更新状态
